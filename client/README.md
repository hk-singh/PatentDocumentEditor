# Patent Reviewer UI

## Layout

Application code is in the `src/` directory.

```
src
├── App.tsx # Main App Shell
├── Document.tsx # Document editor component
├── Editor.tsx # Tiptap Editor
├── LoadingOverlay.tsx # Loading overlay component
├── main.tsx
```

## Running locally

To run locally,

```sh
npm install
npm run dev
```
